Third-Party Dependencies

The LabVIEW project depends on the following third-party VI packages:
	- OpenG Array Library  ; Default installation location: C:\Program Files (x86)\National Instruments\LabVIEW 2021\user.lib\_OpenG.lib\array\array.llb
	- OpenG String Library ; Default installation location: C:\Program Files (x86)\National Instruments\LabVIEW 2021\user.lib\_OpenG.lib\string\string.llb

Use the JKI VI Package Manager (VIPM) tool to install these dependencies.
	- The OpenG dependencies can be installed together by installing the OpenG Toolkit.
